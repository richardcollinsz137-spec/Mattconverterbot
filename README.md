# Matt PDF-to-Text Bot

A Telegram bot built with Python that converts PDF documents to text using `pdfplumber` and `Tesseract OCR`.

## Deployment on Render.com

1. **Create a New Web Service** (or Background Worker) on Render.
2. **Connect your GitHub Repository**.
3. **Runtime**: Python 3
4. **Build Command**: 
   ```bash
   pip install -r requirements.txt
